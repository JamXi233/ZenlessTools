# 安装方法
打开Powershell

输入irm zentools.jamsg.cn/get | iex

回车后将自动安装并启动ZenlessTools

# 注意事项
本工具箱仅适用于PC版绝区零。

⚠️修改游戏画质和帧率可能会影响游戏的稳定性和性能。
